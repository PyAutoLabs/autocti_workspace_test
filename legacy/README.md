# Legacy — Euclid VIS heritage (2022-2023)

This directory preserves, verbatim, the pre-resurrection contents of
`autocti_workspace_test`: Euclid VIS thermal-vacuum (`tvac/`), temporal trap-evolution
(`temporal/`), Euclid pipeline glue (`euclid/`), validation studies (`validation/`),
overview output checks (`overview_output/`) and the era's config (`config_2023/`).

These scripts target the pre-2025 PyAutoCTI API (the removed Plotter object stack,
analysis summing, etc.) and are **not runnable against the current stack**. They are kept
as Euclid VIS calibration history — see the CTI resurrection epic (PyAutoCTI#82) for the
modernization; current integration tests live in `scripts/`.
